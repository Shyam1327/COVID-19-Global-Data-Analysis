# 🌍 COVID-19 Global Data Analysis (Jan 2020 – May 2021)

## 📌 Project Overview
This project presents a comprehensive analysis of COVID-19 data across 277 countries and their respective states/provinces from January 2020 to May 2021. The goal is to uncover trends in confirmed cases, recoveries, and deaths, and to visualize the spread and impact of the pandemic over time.

Using data analysis and visualization techniques, this project highlights key insights that can support public health understanding and decision-making.

---

## 🎯 Objectives
- Analyze global COVID-19 trends across multiple regions  
- Identify peak infection periods and spread patterns  
- Compare confirmed, recovered, and death cases  
- Create meaningful visualizations for better insights  
- Clean and structure complex multi-level datasets  

---

## 🛠️ Tech Stack
- **Python**
- **Pandas** – Data cleaning & manipulation  
- **NumPy** – Numerical computations  
- **Matplotlib** – Data visualization  
- **Seaborn** – Advanced statistical plots  

---

## 📂 Dataset
- Multi-country COVID-19 dataset including:
  - Confirmed cases  
  - Recovered cases  
  - Death counts  
- Data includes country-level and state/province-level records  
- Time range: **January 2020 – May 2021**

---

## 🔍 Data Processing
- Handled missing and inconsistent values  
- Aggregated data at country and global levels  
- Converted time-series data into analyzable formats  
- Structured hierarchical (country/state) datasets

## 📊 Exploratory Data Analysis (EDA)

- Peak period identification
